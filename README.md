🚀 My Portfolio App










A modern Android Portfolio Application built using Kotlin that showcases personal information, projects, skills, resume, and contact details in a clean and professional UI.

The app is designed with a focus on minimal design, smooth navigation, and real-world portfolio presentation.

✨ Features
👨‍💻 Personal Profile Section
Developer introduction
Career objective
Social media links
🛠️ Skills Showcase
Categorized technical skills
Clean card-based UI
📂 Projects Section
Detailed project descriptions
Technologies used
Real-world project highlights
🎓 Education Timeline
Timeline-based academic journey
📄 Resume Viewer
View resume inside the app
Download & share functionality
📝 Feedback System ⭐
Users can submit feedback directly from the app
Feedback data is stored in Firebase Realtime Database / Firestore
Demonstrates real-time backend integration
Enables future analytics & user engagement tracking
📞 Contact Section
Email intent integration
External profile navigation
📱 App Highlights

✅ Smooth fragment navigation
✅ Material-inspired UI
✅ Resume download & share
✅ Firebase feedback submission (real database integration)
✅ Clean modular architecture

🛠️ Tech Stack
Layer	Technology
Language	Kotlin
Platform	Android
UI	XML + Material Components
Backend	Firebase (Realtime Database / Firestore)
Architecture	Fragment Navigation
🧠 App Architecture
User → Home Fragment  
     → Projects / Skills / Resume  
     → Feedback Fragment → Firebase Database  
     → Contact Fragment → External Intents
